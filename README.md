# 📌 Project Name

A Web app for a TicTacToe Game.

## 🚀 Features
- ✨ Auth
- 🚀 Play Game
- 🎨 View Stats
- 🔥 And more??

## 📦 Installation
Make sure you have [Node.js](https://nodejs.org/) installed, then run:

```sh
yarn install
```

## 🏃 Running the Project
To start the development server, run:

```sh
yarn run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.
